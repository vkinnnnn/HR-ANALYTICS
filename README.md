AI-powered workforce intelligence platform that transforms employee lifecycle data into actionable analytics for HR leaders.

## What It Does

Analyzes **2,466 employees** and **11,803 job history records** to surface:

- **Turnover patterns** by department, grade, location, and function
- **Tenure analysis** with retention curves and early-attrition detection
- **Flight risk scoring** via ML (LogisticRegression on career signals)
- **Career progression** tracking with promotion velocity and stuck-employee detection
- **Manager effectiveness** with span of control and revolving-door flags
- **AI-powered chatbot** for natural language workforce queries
- **Executive reports** generated by LLM or data-driven fallback

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | FastAPI + SQLAlchemy 2.0 async + Pandas + scikit-learn |
| Frontend | React 19 + TypeScript + Vite + Tailwind CSS + Recharts |
| LLM | OpenRouter (Nemotron 120B free default) or OpenAI |
| Design | CodeRabbit-inspired dark theme with glass morphism |
| Database | SQLite (pipeline metadata) + in-memory DataFrames (analytics) |

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+

### Backend

```bash
cd backend
pip install -r requirements.txt

# Create .env with your LLM key (optional but recommended)
cat > .env << 'EOF'
LLM_PROVIDER=openrouter
OPENROUTER_API_KEY=your-key-here
OPENROUTER_MODEL=nvidia/nemotron-3-super-120b-a12b:free
EOF

# Start server
python -m uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev  # http://localhost:3000
```

The backend auto-loads CSV data from `wh_Dataset/` on startup.

## Data Source

Three CSV files in `wh_Dataset/`:

| File | Rows | Purpose |
|------|------|---------|
| `function_wh.csv` | 2,466 | Employee master (hire/expire, grade, dept, location) |
| `wh_history_full.csv` | 11,803 | Job change history with manager hierarchy |
| `wh_user_history_v2.csv` | 100 | Enriched subset with position titles |

**Join key:** `PK_PERSON` = `pk_user`
**Manager hierarchy:** `fk_direct_manager` self-references `pk_user`

## AI Chatbot

The platform includes a persistent slide-out AI chatbot accessible from every page:

- **Slide-out panel** (420px) that compresses main content
- **Context-aware** responses based on current page
- **Inline chart rendering** when AI returns visualization data
- **Chart-click integration** — click any chart bar to auto-ask the AI about it
- **Suggested prompts** change per page (dashboard, turnover, careers, etc.)
- **Multi-LLM support** — switch between Nemotron, DeepSeek, Gemini, Qwen, GPT-4o via Settings

### LLM Configuration

Default: **Nvidia Nemotron 120B** via OpenRouter (free tier).

Switch models at runtime via **Settings > AI / LLM Configuration**, or via API:

```bash
# Get current config
curl http://localhost:8000/api/settings/llm

# Switch to DeepSeek
curl -X POST http://localhost:8000/api/settings/llm \
  -H "Content-Type: application/json" \
  -d '{"model": "deepseek/deepseek-chat-v3-0324:free"}'
```

Available free models via OpenRouter:
- `nvidia/nemotron-3-super-120b-a12b:free` (default)
- `meta-llama/llama-3.3-8b-instruct:free`
- `deepseek/deepseek-chat-v3-0324:free`
- `google/gemini-2.0-flash-exp:free`
- `qwen/qwen3-235b-a22b:free`

## Environment Variables

| Variable | Required | Default | Purpose |
|----------|----------|---------|---------|
| `LLM_PROVIDER` | No | `openrouter` | `openrouter` or `openai` |
| `OPENROUTER_API_KEY` | For AI features | — | OpenRouter API key |
| `OPENROUTER_MODEL` | No | `nvidia/nemotron-3-super-120b-a12b:free` | Model ID |
| `OPENAI_API_KEY` | If using OpenAI | — | OpenAI API key |
| `OPENAI_MODEL` | No | `gpt-4o-mini` | OpenAI model |
| `SENTRY_DSN` | No | — | Error monitoring |
| `DATA_DIR` | No | `wh_Dataset/` | CSV data directory |
| `CORS_ORIGINS` | No | `http://localhost:3000` | Additional CORS origins |

**Without any API key:** All analytics work. Chat and reports use data-driven local fallback.

```
    routers/           # 14 route modules
    services/          # Job queue, auth, scheduler, pipeline runners
frontend/
  src/
    pages/             # 14 page components
    components/
      ui/              # Panel, KpiCard, Badge, PageHero, etc.
      layout/          # Sidebar, AmbientBackground
      charts/          # ChartTooltip
      chat/            # ChatTrigger, ChatPanel (slide-out)
    lib/               # API client, utilities
    types/             # TypeScript interfaces
```
